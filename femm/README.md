# VCM 자기 FEA 검증 (FEMM) — 실행 가이드

> 원통형 VCM의 공극 자속밀도 $B_g$ 를 무료 FEA 툴 **FEMM**으로 교차검증.
> (Permeance 해석모델은 이미 논문으로 6% 검증됨 — 이건 $B_g\le0.45$T 가드의 보강 검증.)

## 1. 설치
- https://www.femm.info → Download → **FEMM 4.2** (Windows, 무료).
- 설치 후 실행파일: `C:\femm42\bin\femm.exe`.

## 2. 실행
**방법 A (GUI)**: FEMM 실행 → 메뉴 `File > Open Lua Script...` → `vcm_axi.lua` 선택 → 자동 실행. 결과는 출력창(아래 콘솔)과 `vcm_axi_result.txt`에 기록.

**방법 B (콘솔)**: FEMM 콘솔(Lua)에서
```lua
dofile("C:\\...\\final project\\femm\\vcm_axi.lua")
```

## 3. 검증 순서 (중요)
1. **먼저 `CASE="paper"` (스크립트 상단 기본값)로 실행** → 공극 $B_g$가 **~108 mT** 근처면 FEMM 모델 셋업이 옳다는 뜻(논문 최종설계 재현).
2. 확인되면 스크립트 상단 `CASE="opt"`로 바꿔 재실행 → **우리 최적해(Al, d_c=0.3) 형상의 실제 $B_g$** 확인.
   - 우리 Permeance 모델은 이 형상에서 $B_g$=450 mT(가드 상한)로 계산.
   - FEMM 결과가 ~450 mT 근처면 최적해 신뢰, 더 낮으면(예: 250~350) 추력 과대평가였으니 설계에 마진 반영 필요.

## 4. 모델 개요
- **축대칭 자기정역학**: r=반경, z=축. 자속경로 자석(+z)→중앙요크→공극(반경방향)→측면요크→하단요크.
- 재료: NdFeB($B_r$=1.17T, $\mu_r$=1.05), 요크(선형 $\mu_r$=1400, 해석모델과 동일), 공기. 코일 무전류(자석장만 측정).
- $B_g$ = 중앙요크 높이·중간반경($r_m+t_g/2$)에서 반경방향 $|B_r|$ 평균(21점).
- 개방경계는 `mi_makeABC`(점근경계)로 처리.

## 5. 해석 메모
- 요크를 선형 $\mu_r$=1400으로 둔 것은 해석모델과 1:1 비교 목적. 실재 포화를 보려면 FEMM 라이브러리의 비선형강(`1010 Steel` 등)으로 바꿔 비교 가능.
- 결과를 `[4단계] FEA_검증.md`의 B절 표에 기입하면 검증 마무리.
