## A. 유연기구 모델 — `flexure_model.m`

### 모델
- 토폴로지: Double Compound Parallelogram (8 blade), 운동축 X.
- 운동방향 강성(폐형식): $k_x = 24\,EI/L^3$, $I = b\,t^3/12$ → $k_x = 2E b t^3/L^3$.
- 변위 $x = F/k_x$, 1차 공진 $f_1 = \frac{1}{2\pi}\sqrt{k_x/m_{eff}}$.
- blade 응력 $\sigma = 3E t\,(x/2)/L^2$ (compound라 단 stage당 행정 절반).
- 유효질량 $m_{eff} = m_{stage} + m_{coil} + \tfrac14 m_{inter} + \tfrac13 m_{blade}$ (모드 속도비 반영).

### 검증 (수치)
| 검증 | 방법 | 결과 |
|---|---|---|
| guided blade 강성 | 빔 4×4 강성행렬에서 회전구속 응축 $K(3,3)$ vs $12EI/L^3$ | 오차 $\sim10^{-23}$ (기계정밀도) ✓ |
| DCP 강성 | 직병렬 산술(blade→parallelogram→compound→double) vs $24EI/L^3$ | 오차 0 ✓ |

|---|---|---|---|---|---|---|---|
| Al6061 | 17,040 | 29.3 | 121.3 | 17.0 | 53.2 | 5.18 | OK |
| 스프링강 | 48,000 | 73.9 | 128.2 | 48.0 | 150.0 | 8.00 | OK |

## B. VCM 모델 — `vcm_model.m`

### 모델 (원통형 대칭 moving-coil, Permeance method, 2023 논문 식 1-17)
- Permeance 6종(식 1-6): 자석 $P_m$, 중앙요크 $P_{y1}$, 하단요크 $P_{y2}$, 측면요크 $P_{y3}$, 공극 $P_g$, 누설 $P_l$.
- 자기회로 풀이: 자석을 flux source $\Phi_r=B_r\pi r_m^2$ ∥ $P_m$ 로 보고
  $$\Phi_g = \Phi_r\,\frac{P_{main}}{P_m + P_l + P_{main}},\quad P_{main}=\text{series}(P_g,P_{y1},P_{y3},P_{y2})$$
- 공극 자속밀도 $B_g = \Phi_g/A_g$, $A_g = 2\pi(r_m+t_g/2)h_{y1}$ (평균반경 원통면).
- 권선수(식 10), 평균턴길이 $L_m$(식 12), 유효권선수 $n_{eff}$(식 16), 추력 $F = n_{eff}B_g L_m I$(식 17).
- 저항 $R_c$(식 11), 코일질량 = $\rho_{cu}\,n(\pi/4)d_c^2 L_m$.
- **대칭계수 $n_{sym}=2$**: 논문은 mirror half 모델 → 전체 VCM은 권선수·추력·질량 2배.

### 검증 (논문 최종설계 Ø13×40 재현)
| 항목 | 모델 | 논문 | 오차 |
|---|---|---|---|
| 공극자속 $B_g$ | 119.3 mT | 108.3 mT | +10% |
| 권선수 $n$ | 264 | 304 | −13% |
| 힘상수 $K_f$ | 0.858 N/A | 0.910 N/A | **−6%** |
| 코일질량 | 3.55 g | 4.1 g* | −13% |

\* 논문 코일질량은 보빈 포함. 힘상수 6% 이내 → Permeance 모델 검증 OK (논문 모델-측정 오차 13%보다 우수).

## C. 시스템 연결 (다음 단계로)
- 두 모델의 결합점: **VCM 코일질량 → 유연기구 $m_{eff}$** (공진에 영향), **VCM 추력 $F=K_f\cdot I$ → 유연기구 변위 $x=F/k_x$**.
- 요구: $I\le2$A에서 $x\ge1$mm 이고 동시에 $f_1>100$Hz, $\sigma\le0.3S_y$, 크기·$d_c$ 제약 충족.
- 5단계: 두 모델 결합 + `fmincon`(SQP, multi-start) 으로 **재료 3종 각각 최적화** → 비교·선택.

## D. 잔여/검증 한계
- 유효질량 계수(1/4, 1/3)는 모드형상 근사 — FEA(4단계, 선택)로 공진 교차검증 권장.
- $A_g$ 평균반경 가정으로 $B_g$ +10% — 보수적으로 추력 약간 과대평가 가능, 설계 시 마진 고려.
- VCM 저항/발열 모델은 형상기반 추정(논문 측정 저항이 더 큼) → 발열 제약은 보수적으로.
